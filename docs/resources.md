### Scientific packages
- [Visual Studio Code](https://code.visualstudio.com/): the most useful code editor (latex, python, fortran, mkdown, ...)
- [Feycalc](https://feyncalc.github.io/): a Mathematica package for symbolic evaluation of Feynman diagrams and algebraic calculations in quantum field theory
- [Feynarts](http://www.feynarts.de/): a Mathematica package for the generation and visualization of Feynman diagrams and amplitudes
- [Formcalc](http://www.feynarts.de/formcalc/): a Mathematica package for the calculation of tree-level and one-loop Feynman diagrams
- [MadGraph5_aMC@NLO](https://launchpad.net/mg5amcnlo): a framework that aims at providing all the elements necessary for SM and BSM phenomenology
- [MCFM](https://mcfm.fnal.gov/): a program calculates cross-sections for various femtobarn-level processes at hadron-hadron colliders
- [Pythia8](https://pythia.org/): a program for the generation of high-energy physics collision events
- [HypExp](https://hypexp.hepforge.org/): expand hypergeometric functions around their parameters
- [LHAPDF](https://lhapdf.hepforge.org/):  the standard tool for evaluating parton distribution functions (PDFs) in high-energy physics
- [AMFlow](https://gitlab.com/multiloop-pku/amflow): a proof-of-concept implementation of the auxiliary mass flow method, which can evaluate Feynman loop integrals and/or phase-space integrations, with arbitrary number of loops and external legs, in arbitrary spacetime dimension, at arbitrary kinematic point, to arbitrary precision. 
- [ColorMath](http://home.thep.lu.se/~malin/ColorMath.html):  a Mathematica package for symbolically performing color summed calculations in $SU(N_c)$.
- [HEP packages for Mac](https://github.com/davidchall/homebrew-hep)
